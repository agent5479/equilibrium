# Equilibrium Kinesiology & Nutrition

Static React rebuild of [equilibrium.kiwi.nz](https://www.equilibrium.kiwi.nz/), hosted on GitHub Pages.

**Live site:** https://agent5479.github.io/equilibrium/

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000/equilibrium/

## Scrape content from live site

Re-run the scraper to refresh content from the WordPress source:

```bash
npm run scrape
```

## Build

```bash
npm run build
```

Output is written to `out/` for GitHub Pages deployment.

## Deployment

Pushes to `main` trigger GitHub Actions (`.github/workflows/deploy.yml`) which builds and deploys to GitHub Pages.

Enable Pages in repo settings: **Source → GitHub Actions**.

## Project structure

- `content/` — scraped page and recipe JSON (generated by scraper)
- `public/assets/` — downloaded images and media
- `scripts/scrape/` — site scraper
- `src/app/` — Next.js routes mirroring WordPress URLs
- `src/components/` — layout, page renderer, recipe UI
- `docs/phase-2-booking.md` — booking integration plan

## Phase 1 scope

- Full static replication of public pages, recipes, and categories
- Enfold theme styling (colors, Montserrat font, layout)
- SEO: robots.txt, sitemap.xml, Open Graph metadata, JSON-LD
- Contact/newsletter forms (visual only)

## Phase 2 (planned)

- Google Calendar booking integration via Apps Script
- Functional contact form and newsletter signup
