# Equilibrium Kinesiology & Nutrition

Static React rebuild of [equilibrium.kiwi.nz](https://www.equilibrium.kiwi.nz/), hosted on GitHub Pages.

**Live site:** https://equilibriumhealth.nz/

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000/

## Scrape content from live site

Re-run the scraper to refresh content from the WordPress source:

```bash
npm run scrape
```

## Build

```bash
npm run build
```

Output is written to `out/` for GitHub Pages deployment.

## Deployment

Pushes to `main` trigger GitHub Actions (`.github/workflows/deploy.yml`) which builds and deploys to GitHub Pages.

Enable Pages in repo settings: **Source → GitHub Actions**.

## Project structure

- `content/` — scraped page and recipe JSON (generated by scraper)
- `public/assets/` — downloaded images and media
- `scripts/scrape/` — site scraper
- `src/app/` — Next.js routes mirroring WordPress URLs
- `src/components/` — layout, page renderer, recipe UI
- `docs/phase-2-booking.md` — booking integration plan

## Phase 1 scope

- Full static replication of public pages, recipes, and categories
- Enfold theme styling (colors, Montserrat font, layout)
- SEO: robots.txt, sitemap.xml, Open Graph metadata, JSON-LD
- Contact/newsletter forms (visual only)

## Phase 2 — Booking

Online booking is live at `/bookings/`. Setup instructions:

1. Paste [`google-apps-script/Code.gs`](google-apps-script/Code.gs) into a Google Apps Script project
2. Set Script Properties (calendar + admin + clients sheet) — see [docs/google-apps-script-setup.md](docs/google-apps-script-setup.md)
3. Run `setupClientsSheet()` once
4. Deploy as web app (Execute as: Me, Access: Anyone)
5. Add the web app URL as GitHub Secret `NEXT_PUBLIC_BOOKING_API_URL`
6. Optionally store `ADMIN_PASSWORD` as a GitHub Secret (ops vault only — not used at build time)

## Phase 2 — Contact, newsletter, admin

- Contact and newsletter forms on `/contact/` write to the Clients sheet and notify Patricia
- Client list + email composer at `/admin/` (password from Apps Script Script Property `ADMIN_PASSWORD`)
